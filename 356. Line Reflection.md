# Line Reflection
Given n points on a 2D plane, find if there is such a line parallel to y-axis that reflect the given points symmetrically, in other words, answer whether or not if there exists a line that after reflecting all points over the given line the set of the original points is the same that the reflected ones.

Note that there can be repeated points.


## 思路
如果所有的点对于一条平行于y轴的线对称, 那么他们两两x轴的加和就是线的x坐标. 所有这道题就是先遍历一遍找到最大最小x值, 找到对称轴的坐标然后验证其他的点.
trick是如何验证, 用hashset记录所访问过的点, set中可以放入访问的点的int array的hashcode或者把int array变成string放入set

time O(n) space O(n)

### 代码

```
class Solution {
    public boolean isReflected(int[][] points) {
        int max_x = Integer.MIN_VALUE;
        int min_x = Integer.MAX_VALUE;
        Set<Integer> set = new HashSet<>();
        for (int[] point : points) {
            min_x = Math.min(min_x, point[0]);
            max_x = Math.max(max_x, point[0]);
            set.add(Arrays.hashCode(point));
        }
        int mid = min_x + max_x;
        for (int[] point : points) {
            if (!set.contains(Arrays.hashCode(new int[]{mid-point[0], point[1]}))) {
                return false;
            }
        }
        return true;
    }
}
```
